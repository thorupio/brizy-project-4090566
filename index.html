FAQ

Is Undo a complete backup of my Shopify store?

Undo is not a complete backup and recover solution meant to address disaster situations where you are in need of a complete recovery of your shop's assets including orders and customers. For this, you need the help of Shopify Support or a dedicated backup solution.

Undo is a tool for peace of mind with regard to your content setup - products, images, collections, pages, blogs, custom fields etc. - as well as a productivity tool for everyday use, where you need to quickly restore selected objects to a previous state - for whatever reason like a small mistake, a bad bulk import, a buggy app or even a disgruntled employee making a mess of your setup.

What happens when I install Undo?

When your first install Undo in your Shopify store, it establishes a "baseline" for the various supported Shopify objects.

A baseline is an initial backup of all relevant information needed to recover an object such as product titles, descriptions, images etc. so you may later return to the state of these objects if needed. Once this initial baseline is complete, Undo continually monitors changes made in your Shopify store that affect the state of the backed-up information.

Changes are recorded within Undo as "undo points" per individual object and you can use Undo's functionality to return the any of these objects' current state at the time of recording.

Does Undo work with my custom fields?

Yes, Undo supports metafields, which is the underlying technology for Custom Fields provided by different metafield apps including our own Accentuate Custom Fields.

While backing up and restoring metafields works the same across different metafield apps, extra integration points has been made between Undo and Accentuate Custom Fields to make sure everything works smoothly when it comes to detecting changes and presenting information within Undo.

Also, Undo backs up your key objects' media content uploaded in Accentuate Custom Fields for a true time-versioned experience.
